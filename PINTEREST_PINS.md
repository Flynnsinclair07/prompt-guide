## Canva production notes

- Open one of the existing pins (`pins/pin-01-resume.png`) in Canva → "Use as template" → swap text per the spec blocks above. Don't rebuild from scratch — the existing design is the brand.
- Font: Arial Black for headlines, Arial Bold for sub-line and benefits. Both system fonts on macOS, free in Canva.
- Checkmark glyph: ✓ (U+2713). Canva's auto-suggested check icons drift from the existing pins' look — copy-paste the actual character.
- Export: PNG, 1000×1500, "Best" quality. Don't use Canva's "share to Pinterest" — upload directly to Pinterest so you control the description and alt text.

---

## Pinterest platform settings (one-time)

- **Pin type**: Standard pin (not Idea Pin — Idea Pins don't accept outbound links).
- **Board**: AI Prompts for Job Seekers (existing).
- **Title field on Pinterest**: paste the headline text from the pin (e.g. "Resume bullets that don't suck — free ChatGPT prompt").
- **Description field**: paste the description block above.
- **Alt text field**: paste the alt text block above.
- **Destination link**: paste the outbound link from the pin's spec.

---

## Voice / pin-copy notes (don't paste, for consistency)

- Headlines are imperative or declarative, never aspirational ("Become…", "Unlock…" — kills CTR on Pinterest).
- Numbers in headlines beat words ("90 seconds", "$29", "60 seconds", "300 characters").
- "Free" in headline only on free-prompt pins (6–12). Bundle pins (13–15) lead with the price as the anchor, not "free."
- Benefits are 3 lines, ≤4 words each. The existing pins (`generate_pins.py:18-25`) do this — match it.
- No emojis in pin headlines or descriptions. Hashtags only at the end of descriptions.
