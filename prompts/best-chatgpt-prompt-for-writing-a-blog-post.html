Best ChatGPT Prompt for Writing a Blog Post

Get a complete, SEO-optimized blog post — with headlines, structure, and keywords — ready to publish.

The Prompt 
You are an experienced content writer and SEO specialist. Write a blog post for me.

Topic: [WHAT THE POST IS ABOUT]
Target keyword: [THE MAIN KEYWORD YOU WANT TO RANK FOR]
Target audience: [WHO IS READING THIS]
Word count: [e.g. 1000, 1500, 2000]
Tone: [professional / conversational / witty / authoritative]
Goal: [educate / sell / generate leads / build authority]

Requirements:
- Write a compelling headline with the target keyword
- Include an engaging intro that hooks the reader in the first 2 sentences
- Use H2 and H3 subheadings to break up the content
- Naturally include the target keyword 3-5 times (don't stuff it)
- Include a clear call to action at the end
- Write in short paragraphs (2-3 sentences max)
- Add a meta description (under 160 characters)
- Suggest 3 internal linking opportunities (related topics I should write about next)
- Make it scannable — someone skimming should get the main points from headers alone

How to Use This Prompt

  1. Copy the prompt and fill in your topic and keyword
  2. If you don't know your target keyword, ask the AI first: "What keywords should I target for a blog post about [topic]?"
  3. Specify the word count — longer posts (1500+) tend to rank better on Google
  4. Paste into ChatGPT or Claude
  5. Edit the output to add your personal experience and voice — Google rewards originality

Never publish AI content without editing it. Add your own stories, data, and opinions to make it unique.

This guide is different. We'll walk through exactly how to pick an idea, validate it in 48 hours, and make your first sale — even if you've never done this before.

Tips to Get Better Results

  • Research first. Google your keyword and read the top 3 results. Tell the AI: "The top results cover X and Y. Make my post cover Z that they missed."
  • Add data. After getting the draft, say "Add 3-5 relevant statistics with sources to strengthen the arguments."
  • Get an outline first. If the full post isn't great, ask for just an outline first, approve it, then say "Now write the full post from this outline."
  • Humanize it. Tell the AI: "Rewrite the intro as if you're talking to a friend over coffee." This fixes the robotic tone.
