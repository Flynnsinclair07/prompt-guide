Best ChatGPT Prompt for Real Estate Listings

Write property descriptions that make buyers schedule a showing — in under a minute.

The Prompt
You are a top-producing real estate agent and copywriter. Write a compelling MLS listing description for this property.

Property details:
- Address/neighborhood: [LOCATION]
- Type: [single family / condo / townhouse / multi-family]
- Bedrooms: [NUMBER] | Bathrooms: [NUMBER]
- Square footage: [NUMBER]
- Year built: [YEAR]
- Lot size: [SIZE]
- Price: $[AMOUNT]
- Key features: [LIST 5-8 STANDOUT FEATURES, e.g. updated kitchen, hardwood floors, pool, garage, etc.]
- Recent upgrades: [anything renovated or new]
- Neighborhood highlights: [schools, parks, walkability, nearby amenities]

Target buyer: [e.g. first-time buyers, families, investors, downsizers]

Requirements:
- Open with a hook that creates emotion, not just facts
- Keep it under 250 words (MLS has character limits)
- Highlight the top 3 selling points in the first two sentences
- Use sensory language — help buyers picture living there
- Include a closing line that creates urgency
- Avoid cliches like "move-in ready" and "won't last long" — find fresher language
- Follow Fair Housing guidelines — no discriminatory language

How to Use This Prompt

  1. Fill in every property detail — the more specific, the better the description
  2. Include the target buyer so the AI tailors the language
  3. Paste into ChatGPT or Claude
  4. Review for accuracy — never publish details you haven't verified
  5. Copy into your MLS system

Example Output

Morning light pours through the oversized windows of this 3-bed, 2-bath craftsman in the heart of Willow Creek — the kind of home you walk into and immediately stop looking.

The chef's kitchen was fully remodeled in 2024 with quartz counters, soft-close cabinetry, and a 6-burner gas range. Original hardwood floors run throughout the main level, and the primary suite opens to a private deck overlooking a mature, landscaped backyard.

Two blocks from Lincoln Elementary (rated 9/10) and a 10-minute walk to downtown shops and restaurants. The detached two-car garage has been insulated and wired — ready for a workshop, studio, or home gym.

Open house this Saturday. Bring your highest expectations.

Tips to Get Better Results

  • Lead with lifestyle, not specs. "Morning light pours through" beats "3BR/2BA with large windows."
  • Ask for multiple versions. Say "Give me a luxury version and a first-time-buyer version" for different audiences.
  • Add the neighborhood story. Tell the AI what makes the area special — walkability, schools, vibes.
  • Request social media versions. After the MLS copy, say "Now write a 50-word Instagram caption for this listing."
